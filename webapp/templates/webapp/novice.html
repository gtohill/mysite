Novice Course

Start Developing!

Learn to create fun and interactive video games using Scratch!


Core Concepts: variable assignment, loops, cartesian coordinates, math (linear and algebra), conditional statements, logic.


Python is the ideal language to learn for first time coders.

This course is for students who have completed our Beginners Course (Scratch) or have some experience with basic programming concepts like, loops, conditional statements, variable assignments, functions, and methods.

The Python programming language was created in 1991 by Guido van Rossum. Python is used by YouTube, Google, NASA, CERN, Industrial Light & Magic, and Firaxis Games. As of 2017, Python is rated the fourth most popular programming language.

Python is used for website development, scientific computing, and artificial intelligence. It is considered the easiest language to learn for first time programmers. Python was designed to be easily readable and uses English keywords.

In the Novice course, students will create a real functioning game using Python + Pygame. The focus of this course is to teach students more complex uses of loops, conditional statements, functions, methods, and algorithms. Students will be introduced to object-oriented design, encapsulation, decomposition, and inheritance.
