Recommended Age: 14+


This course is for older students who have some experience with a dynamically typed programming language and good typing skills.

The intermediate course build on the concepts learned in the novice course. The Intermediate course focus' on using computational thinking to solve programming challenges. We will develop more complex games with a focus on game physics, basic Artificial Intelligence (Enemy AI), object oriented design, data structures, and linear algebra.
