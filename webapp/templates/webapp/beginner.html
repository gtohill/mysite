Recommended Age: 8+

Core Concepts: variable assignment, loops, cartesian coordinates, math (linear and algebra), conditional statements, logic.


Scratch is a programming language that has influenced many other programming environments and is now considered a standard for introductory coding experiences for children.

With Scratch, children can create their own interactive stories, games and animations. Scratch is developed by MIT Media Lab to help children learn to think creatively, reason systematically and work collaboratively.

Scratch is a very popular and widely used to teach kids basic programming concepts. Scratch has more than 22 million registered members on its online community and more than 26 million shared projects, with roughly 25,000 new members and 30,000 new projects every day.

The popularity of Scratch provides your child with the opportunity to continue to explore programming challenges outside of the class room.

Guided by our instructors, students will not only create a real functioning game from "Scratch", they'll practice the fundamental coding concepts required in any language, learn how to think logically, and have fun while doing it.
